# rate-limit-api-node-express

## Features

- Use node.js v12
- Use express v4.16

## Install

```bash
$ git clone <reop>
```

## Commands

- Before development

```
$ npm install
```

- Start

```
$ npm start
```

- Test

```
$ npm run test
```

- Build

```
$ docker-compose build
```

- Run docker

```
$ docker-compose up -d
```
